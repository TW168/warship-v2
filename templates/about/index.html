About Warship

What is Warship?

Warship (Warehouse and Shipping Management System) is a professional internal tool for managing warehouse inventory, shipping logistics, and technical support requests in a unified interface.

Built on a modern Python stack — FastAPI, SQLAlchemy, and Bootstrap 5 — Warship is designed for reliability, speed, and ease of use across desktop and mobile devices.

Technology Stack
Layer Technology Purpose
Backend FastAPI REST API and HTML template serving
ORM SQLAlchemy Database access and query building
Database MySQL Persistent data storage
Frontend Bootstrap 5 Responsive UI components
Interactivity HTMX Partial page updates without full reloads
